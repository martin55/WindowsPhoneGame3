Methods
  NameDescription
Application_Activated
Code to execute when the application is activated (brought to foreground) This code will not execute when the application is first launched.
Application_Closing
Code to execute when the application is closing (e.g. user hit Back) This code will not execute when the application is deactivated.
Application_Deactivated
Code to execute when the application is deactivated (sent to background) This code will not execute when the application is closing.
Application_Launching
Code to execute when the application is launching (e.g. from Start) This code will not execute when the application is reactivated.
Application_UnhandledException
Code to execute on Unhandled Exceptions.
