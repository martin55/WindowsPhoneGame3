GamePage Fields A Sandcastle Documented Class Library

Fields
  Name Description
_contentLoaded
accelerometer
Representation of the physical accelerometer sensor.
camera
Game camera; at fixed height over the board, movable in different directions, always following the main character.
contentManager
Content manager which fetches media from the LibContent project.
crates
A collection of game object bodies for crates, movement hindering and annoying things.
currentPlayerAnimationDelay
Current delay for the shark's movement animation.
gameWorld
Game world that holds all objects and manages all the physics and collisions.
humans
A collection of game object bodies for humans, game's "antagonists" and prey.
isGameOver
A value indicating whether the game is over.
particleEngine
Particle generator.
playerAnimationDelay
Delay for the shark's movement animation (optimization aspects).
playerCurrentFrame
Current frame for the shark's movement animation.
points
An amount of points the player has collected throughout the current level.
pools
A collection of game object bodies for pools, timer replenishers for the shark.
runFrameHeight
Height of the running fish's single animation frame.
runFrameWidth
Width of the running fish's single animation frame.
shark
Game object body for the shark, game's "protagonist".
spriteBatch
Textures for all game objects.
sprites
Collection of game sprites for corresponding bodies.
tiles
Collection of sprites for the map.
timeLeft
Time left until the shark suffocates.
timer
Main game timer for update events.
timeSinceEatingSound
An amount of time that should pass before the eating sound effect can be played again.
timeSinceGameOver
An amount of time that should pass before the game over screen fades.
timeSinceTrapSound
An amount of time that should pass before the trap sound effect can be played again.
timeSinceWaterSound
An amount of time that should pass before the water sound effect can be played again.
traps
A collection of game object bodies for traps, shark insta-killers.
walls
A collection of game object bodies for walls, movement restraining invisible barriers.
